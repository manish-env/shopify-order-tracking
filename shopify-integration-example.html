🛒 Shopify Add to Cart Button Control

Disable Add to Cart buttons based on order status without modifying your theme

📋 Integration Methods

Method 1: Direct JavaScript Integration

Add this code to your Shopify theme's theme.liquid file or any template:

<script> // Shopify Add to Cart Button Control (function() { // Replace with your actual API URL const API_URL = 'https://your-api-domain.workers.dev'; // Function to check order status and control buttons async function checkOrderAndControlButtons(orderNumber, email) { try { const response = await fetch(`${API_URL}/shopify-button-control`, { method: 'POST', headers: { 'Content-Type': 'application/json', }, body: JSON.stringify({ orderNumber: orderNumber, email: email }) }); if (response.ok) { const script = await response.text(); // Execute the returned JavaScript eval(script); } else { console.error('Failed to load button control script'); } } catch (error) { console.error('Error loading button control:', error); } } // Call this function when you want to check order status // Example: checkOrderAndControlButtons('12345', 'customer@example.com'); // Make function globally available window.checkOrderStatus = checkOrderAndControlButtons; })(); </script>

Method 2: Dynamic Script Loading

Load the script dynamically based on URL parameters or user input:

<script> // Load button control script dynamically function loadButtonControlScript(orderNumber, email) { const script = document.createElement('script'); script.src = `${API_URL}/shopify-button-control`; script.onload = function() { // Script will automatically execute and control buttons }; document.head.appendChild(script); } // Example usage if (window.location.search.includes('order=')) { const urlParams = new URLSearchParams(window.location.search); const orderNumber = urlParams.get('order'); const email = urlParams.get('email'); if (orderNumber) { loadButtonControlScript(orderNumber, email); } } </script>

Method 3: Shopify App Integration

For more advanced integration, you can create a Shopify app that injects this functionality:

// In your Shopify app's frontend code function injectButtonControl() { const script = document.createElement('script'); script.textContent = ` // Your button control logic here // This would be generated by your app's backend `; document.head.appendChild(script); } // Call when app loads document.addEventListener('DOMContentLoaded', injectButtonControl);

⚙️ Configuration

API Endpoints:
  • POST /shopify-button-control - Returns JavaScript for button control
  • POST /track - Returns JSON with order status and button state
  • POST /button-control - Returns HTML page with embedded JavaScript
Important: Replace https://your-api-domain.workers.dev with your actual API domain after deployment.

📱 Usage Examples

Basic Usage

// Check order status and control buttons window.checkOrderStatus('12345', 'customer@example.com');

With URL Parameters

// Check for order in URL parameters const urlParams = new URLSearchParams(window.location.search); const orderNumber = urlParams.get('order'); const email = urlParams.get('email'); if (orderNumber) { window.checkOrderStatus(orderNumber, email); }

Manual Control

// After script loads, you can manually control buttons if (window.shopifyOrderTracking) { window.shopifyOrderTracking.disableButtons(); window.shopifyOrderTracking.enableButtons(); }
